**Задание:**

Разработать Fabric мод под Minecraft версии 1.21.8, API последней версии. В качестве системы сборки использовать Gradle 8.14.

1. Простой Screen с полем ввода текстового сообщения и кнопкой отправки.
2. По нажатии на кнопку клиентом необходимо отправлять закодированное в Protobuf 3 сообщение
message Message {
string text = 1;
}
3. На сервере декодировать данное сообщение и записывать его содержимое в таблицу Postgres следующего вида:
CREATE TABLE messages (
id SERIAL PRIMARY KEY,
uuid UUID NOT NULL,
text VARCHAR(256) NOT NULL
);
Где id — уникальное ID записи;
uuid — уникальный ID игрока, который отправил данное сообщение;
text — содержимое Protobuf сообщения = поле text.

Для GUI можно использовать LibGUI или стандартное API Minecraft — на выбор, что для вас быстрее.
Для Protobuf использовать официальную Google библиотеку и плагин для Gradle.
Будет плюсом, если для работы с БД примените Hibernate с JPA Repository или сможете объяснить, как применить.
Маппинги использовать официальные Mojang.
